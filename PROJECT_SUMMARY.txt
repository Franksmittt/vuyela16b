DESIGN SYSTEM
================================================================================
Color Palette:
- Primary Background: #1a1a1a (Dark)
- Secondary Background: #2a2a2a (Medium Dark)
- Tertiary Background: #3a3a3a (Lighter Dark)
- Accent Gold: #FFD700 (Primary brand color)
- Text Primary: White (#FFFFFF)
- Text Secondary: Gray-300 (#D1D5DB)
- Text Muted: Gray-400 (#9CA3AF)
- Light Background: White (#FFFFFF) for alternating sections

Typography:
- Primary Font: Inter (--font-inter)
- Display Font: Poppins (--font-poppins)
- Font weights: 400, 500, 600, 700

Layout:
- Container: max-w-full with responsive padding (px-4 sm:px-6)
- Sections: py-12 sm:py-16 md:py-20
- Responsive breakpoints: sm (640px), md (768px), lg (1024px)
- Grid layouts: 1 column mobile, 2 columns tablet, 3-4 columns desktop

Styling Approach:
- Tailwind CSS v4 with canonical syntax
- Custom CSS variables for theming
- Dark theme optimized
- Light/dark mode support via next-themes

Animations:
- Framer Motion for page transitions
- Spring animations (stiffness: 400, damping: 25)
- Staggered animations for lists
- Hover effects and transitions
